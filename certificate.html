Welcome to the Development Certificate Process

For smooth integration and stakeholder satisfaction, Triple Play Pay has built this quick certification process that utilizes a test API Key and specified tests in our Sandbox environment to ensure that developers have the resources and know-how to properly utilize our endpoints, requests, and responses for successful transactions and user experiences.

Work through these steps for your successful integration of Triple Play Pay’s API.

You will accomplish:

  1. Familiarizing yourself with the Developer Documentation and GitHub repository
  2. Integrating the API by using a Sandbox test API key
  3. Submitting our required test data
  4. Receiving a certificate of completion

Setup

To get an API Key and for access to our Sandbox testing environment, your organization must enroll as an ISV or a merchant with Triple Play Pay Inc. We are structured to work with ISVs who serve many (even thousands of) merchants as well as directly with individual merchants of any size.

Once your business has enrolled with Triple Play Pay:

  1. Ensure that you have an account with Sandbox access. An admin user at your organization must grant Sandbox access.
  2. Retrieve your business's API key in the Processing Settings.
  3. Integrate the API.
  4. Perform the tests listed in the Tests section of this page.
  5. View your business's certificate in the portal here. This is visible for all admin users in your organization to be aware of what your team has coded up to.
